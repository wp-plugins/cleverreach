Easily integrate a CleverReach Sign-Up form in your website. Supports widget, shortcode, comment integration and template function.

== Description ==
Connect your WordPress website to your [CleverReach Newsletter](http://www.cleverreach.com/?rk=74089ojcjrsrb) account.

= Drag & Drop Form Builder =

* Get custom fields from your CleverReach list
* Build your individual Signup-Form via Drag & Drop
* Modify all labels and descriptions
* No coding skills required

= Comment Form Integration =

Display a checkbox in your comment form to allow users to subscribe to your newsletter

= more Features =

* Double-Opt-In
* Detects duplicate entries
* AJAX form submission





= Integrate CleverReach in your Contact form =
Have a look at the [NinjaForms CleverReach plugin](http://etzelstorfer.com/ninja-forms-cleverreach/) to send any contact form submissions directly to your CleverReach lists.

= Free Beautiful HTML Mails for your WordPress site =
With [WP HTML Mail](http://wp-html-mail.com/) you can create beautiful responsive mail templates for your WordPress mails without any coding.

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
